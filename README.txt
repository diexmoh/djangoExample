Para utilizar la aplicación es necesario abrir una terminal y 
    ejecutar el comando "python manage.py runserver" (sin comillas) 
    en la ruta "../proyecto_django/proyecto_django/"
    una vez que este corriendo el servidor abrimos la url indicada en nuestro navegador.
Esta aplicación fue desarrollada con Python 2.7.12 y Django 1.11.2
